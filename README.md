## Run Locally

Clone the project

```bash
  git clone [https://link-to-project](https://github.com/ethsam/react-simple-slider.git)
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies and update

```bash
  npm install
  npm update
```

Start the server

```bash
  npm start
```
