## Run Locally

Clone the project

```bash
  git clone https://github.com/ethsam/react-simple-slider.git
```

Go to the project directory

```bash
  cd react-simple-slider
```

Install dependencies and update

```bash
  npm install
  npm update
```

Start the server

```bash
  npm start
```

or

```bash
  NODE_OPTIONS=--openssl-legacy-provider npm start
```
